# CRO Rules

Primary CTA must always be visible above the fold.

Maximum one primary CTA.

Every landing page must answer within 5 seconds:

- What is this?
- Who is it for?
- Why should I care?
- What do I do next?

Every landing page should include:

- Social proof
- Trust
- Benefits
- FAQ
- Clear CTA
- Mobile optimization

Buttons should use action verbs.

Good:

- Start Free Trial
- Book Demo
- Apply Now

Bad:

- Learn More
- Read More
- Continue

Every page should reduce friction.

Always measure:

- Click CTA
- Scroll
- Form Start
- Form Submit

Never add unnecessary UI.

If a section doesn't increase conversion remove it.
